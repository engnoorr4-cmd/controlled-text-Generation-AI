# Controlled Text Generation for Product Descriptions

This project is a system that generates product descriptions based on user input.

## Features
- Generate multiple descriptions
- Include required words
- Avoid forbidden words
- Evaluate results
- Select best output

## Technologies
- Python
- Streamlit

## How it works
The user enters product details, and the system generates descriptions and evaluates them.
